OWASP Broken Web Applications

This is the VM for the Open Web Application Security Project (OWASP) Broken Web Applications project. It contains many, very vulnerable web applications, which are listed below. More information about this project can be found at http://www.owaspbwa.org/.

For details about the known vulnerabilities in these applications, see http://sourceforge.net/apps/trac/owaspbwa/report/1.

!!! This VM has many serious security issues. We strongly recommend that you run
it only on the "host only" or "NAT" network in the virtual machine settings !!!
